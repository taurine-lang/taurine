# Заметки о релизе Taurine v1.0.0

**Дата релиза:** 16 марта 2026 г.

## 🎉 Стабильный релиз!

Мы рады объявить первый стабильный релиз языка программирования Taurine — быстрого, встраиваемого скриптового языка, который сочетает простоту Lua с современными функциями и производительностью Rust.

## ✨ Что нового

### Функции языка

#### let/const объявления
```taurine
// Изменяемая переменная
let x = 10
x = 100

// Константа (нельзя изменить)
const PI = 3.14
// PI = 3  // Ошибка!
```

#### Интерполяция строк (f-строки)
```taurine
let name = "Taurine"
let version = "1.0"
print(f"Привет от {name} v{version}!")
```

#### Multi-Return функции
```taurine
function divmod(a, b) {
    return a / b, a % b
}

let {q, r} = divmod(10, 3)
print(f"Частное: {q}, Остаток: {r}")
```

#### break/continue в циклах
```taurine
for i in 1..10 {
    if i == 5 { break }
    if i == 3 { continue }
    print(i)
}
```

#### Nil-safe операторы
```taurine
let obj = { name: "тест" }
print(obj?.name)      // "тест"
print(obj?.missing)   // nil (без ошибки)

let nilObj = nil
print(nilObj?.prop)   // nil
```

#### JSON-стиль таблиц
```taurine
// Оба стиля теперь поддерживаются
let obj1 = { name: "тест", value: 42 }  // JSON стиль
let obj2 = { name = "тест", value: 42 }  // Legacy стиль
```

### Улучшения производительности

- **В 2-3 раза быстрее** доступ к переменным с `Rc<RefCell<>>` вместо `Arc<Mutex<>>`
- **Constant folding** оптимизатор для вычислений во время компиляции
- **Dead code elimination** для недостижимого кода

## 📦 Установка

```bash
# Собрать из исходников
cargo build --release

# Запустить скрипт
./target/release/taurine script.tau

# Запустить REPL
./target/release/taurine --repl

# С оптимизациями
./target/release/taurine --optimize script.tau
```

## 🔄 Руководство по миграции

### С v0.x на v1.0

#### Объявления переменных
```taurine
// Старый (всё ещё работает)
loc x = 10

// Новый (рекомендуется)
let x = 10
const PI = 3.14
```

#### Синтаксис таблиц
```taurine
// Старый (всё ещё работает)
let obj = { name = "тест" }

// Новый (рекомендуется)
let obj = { name: "тест" }
```

#### Менеджер пакетов
Менеджер пакетов был удалён. Используйте модули стандартной библиотеки напрямую:

```taurine
// Импорт стандартной библиотеки
import "std/json.tau" as json
import "std/crypto.tau" as crypto
```
